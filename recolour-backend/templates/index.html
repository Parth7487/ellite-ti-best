💡 Tips for best results
📸 Use studio or dark background shots — the algorithm focuses on the dominant paint colour and naturally ignores neutral/dark backgrounds.
⬛ Carbon parts: Raw carbon fibre won't change at all — it's too dark and desaturated to trigger the hue mask. Only coloured accent sections change.
🎯 One colour per image works best. If a bumper has both red and orange accents, the most dominant one is targeted. Use Batch mode to preview multiple colours at once.
🔁 Batch mode — switch to Batch in the top toggle, add multiple colours, and generate all variants from one image with a single click. Great for client colour presentations.
🪞 Floor reflections are auto-managed — the bottom 20% of the image is excluded from recolouring, and the fade zone above gets only a 7% subtle tint.
💰 Zero cost, unlimited. No API used when Keep Background is ON. Pure local processing on your machine — as fast as 0.5 seconds per image.
